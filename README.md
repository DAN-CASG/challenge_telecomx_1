# Telecom X – Análisis de Evasión de Clientes (Churn) - PARTE 1

## Resumen del Proyecto

Este proyecto tiene como objetivo analizar los datos de clientes de **Telecom X** para comprender los factores asociados a la **evasión de clientes (churn)**, es decir, cuando un cliente decide cancelar o dejar de utilizar los servicios de la empresa. El análisis permite identificar patrones en el comportamiento de los clientes y generar información útil para mejorar las estrategias de retención.

### ¿De qué trata el proyecto?

El proyecto se centra en el análisis de un conjunto de datos que contiene información sobre los clientes de Telecom X, incluyendo características como:

* Tipo de contrato.
* Servicios contratados.
* Método de pago.
* Antigüedad del cliente en la empresa.
* Cargos mensuales y cargos totales.
* Estado de evasión del cliente (si abandonó o no el servicio).

A partir de estos datos, se busca comprender qué variables pueden estar relacionadas con una mayor probabilidad de abandono del servicio.

### ¿Qué busca el análisis?

El análisis tiene como finalidad:

* Explorar la estructura del conjunto de datos.
* Identificar valores nulos, errores o inconsistencias.
* Limpiar y transformar los datos para mejorar su calidad.
* Preparar un dataset confiable para análisis exploratorios y modelos predictivos de churn.

### ¿Cómo usar el proyecto?

Para utilizar el proyecto se deben seguir los siguientes pasos:

1. Descargar o clonar el repositorio del proyecto.
2. Abrir el archivo **Telecom_X_parte_1.ipynb** en un entorno compatible con Jupyter Notebook (Jupyter Notebook, JupyterLab o Google Colab).
3. Ejecutar las celdas del notebook en orden para:

   * Cargar los datos.
   * Realizar la limpieza y transformación del dataset.
   * Generar el archivo final de datos procesados.

El dataset resultante puede utilizarse posteriormente para realizar análisis exploratorios más profundos, visualizaciones o modelos de predicción de evasión de clientes.
